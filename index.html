Some basic alchemy knowledge:

  • git --version - reveals the truth about Git's version
  • git init - creates something new called repository
  • git status - reveal the status of the potion
  • git add (.)/(file name) - adds all ingredients/an ingredient
  • git commit -m + "description" - records your sins (actions/changes)
  • git --help - get some help, okay?
  • git log - the story of your sins (commitements)
  • git clone - copies your (r)ecipe
  • git push origin main/branch's name - sends your potions to the sky
  • git branch - shows all the branches of your witchcraft
  • git checkout -b + *name* - creats a new branch of witchcraft called *name*
  • git checkout + branch's name - moves you to another cabinet (branch)
